# Expense Tracker (Machine Learning)

This project is a machine learning based expense tracker that preprocesses expense data,
trains a model, and uses saved models for prediction.

## Features
- Expense data preprocessing
- Machine learning model training
- Saved scaler and model using pickle
- CSV based dataset handling

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## How to Run
1. Clone the repository
2. Install dependencies
3. Run `expense_training.py`
4. Use `expense_ml.py` for predictions
